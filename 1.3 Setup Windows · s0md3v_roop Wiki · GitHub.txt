  1.3 Setup Windows

Jump to bottom <#wiki-pages-box>
Henry Ruhs edited this page Aug 18, 2023 · 1 revision <https://
github.com/s0md3v/roop/wiki/1.3-Setup-Windows/_history>


      Python

<#python>

|winget install -e --id Python.Python.3.10
|


      PIP

<#pip>

|python -m ensurepip
|


      GIT

<#git>

|winget install -e --id Git.Git
|


      FFmpeg

<#ffmpeg>

|winget install -e --id Gyan.FFmpeg
|

Reboot your system in order for FFmpeg to function properly.

|shutdown /r
|


      Toolset

<#toolset>


        Microsoft Visual C++ 2015 Redistributable

<#microsoft-visual-c-2015-redistributable>

|winget install -e --id Microsoft.VCRedist.2015+.x64
|


        Microsoft Visual Studio 2022 build tools

<#microsoft-visual-studio-2022-build-tools>

During installation, ensure to select the Desktop Development with C++
package.

|winget install -e --id Microsoft.VisualStudio.2022.BuildTools --override "--wait --add Microsoft.VisualStudio.Workload.NativeDesktop --includeRecommended"
|
